PROJETO DE CURSO — INTELIGÊNCIA ARTIFICIAL, ESTÉTICA E FUTURO

Resumo:
Curso online autoral, modular e interdisciplinar, combinando teoria crítica, técnica prática, estética visual e narrativa. O conteúdo é pensado para ser independente e rizomático, com foco em inteligência artificial, mídia virtual e cultura digital — misturando minicursos, vídeo-ensaios e material didático de apoio.

Público-alvo:
Pessoas com formação ou interesse avançado em arte, tecnologia, filosofia, design, comunicação, machine learning e áreas correlatas. Não é pensado para o público leigo geral — mas para quem já procura profundidade conceitual e técnica.

Formato dos módulos:
As aulas terão formatos variados — alguns em estilo vlog, outros como documentário narrativo. Haverá módulos teóricos (ex: Baudrillard, cibernética, ética da IA) e módulos práticos (ex: ComfyUI, LoRA, pointclouds, pipelines de geração). Cada módulo será independente, mas interligado a outros por referências cruzadas — uma navegação em rede, não linear.

Interface e estética:
A identidade visual será toda inspirada em terminais antigos estilo MS-DOS (fundo preto, tipografia verde, estrutura monoespaçada), com responsividade total para mobile. A interface do site simulará um terminal onde os módulos e comandos serão acessados como se estivéssemos navegando por um sistema.

Infraestrutura:
- Front-end: React com Tailwind (ou similar)
- Frameworks possíveis: Next.js, Astro
- Hospedagem: Vercel ou Netlify
- Estilo visual: terminal hacker / retro CLI
- Deploy leve e estático para MVP
- Backend: nenhuma necessidade no início
- Possível integração futura com Firebase Auth ou LMS open source

Funcionalidades principais no MVP:
- Página inicial com prompt de entrada (simulando terminal)
- Listagem dos módulos com títulos e descrições resumidas
- Links para vídeos (Vimeo/YouTube não listado)
- Estilo visual consistente com a proposta hacker/retro
- Site leve, acessível e 100% mobile-ready
- Possível versão PT/EN (ou legendas nos vídeos)

Funções extras (futuro):
- Captura de e-mails
- Sistema de acesso por token/manual
- Área de comunidade (fórum leve ou Discord)
- Progresso ou trilhas customizadas (se fizer sentido)

Objetivo:
Construir uma plataforma autoral e diferenciada — estética, funcional e conceitualmente — para apresentar o curso como uma obra em si, articulando arte, código, teoria e técnica.
